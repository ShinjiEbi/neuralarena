# neuralarena-
PWA de combat IA évolutive. Joue contre une IA neuronale (MLP 16→20→14→6) dans Arena Archer, entraîne-la par neuroévolution à chaque partie, stocke les cerveaux en local (IndexedDB) et partage-les via GitHub. Système distribué : chaque joueur entraîne l'IA, les meilleurs cerveaux sont partagés. 🧠⚔️
